# Setup Instructions

1. Set up Python venv and start.
```shell
python -m venv venv
activate
```
2. Download dependencies
```shell
pip install -r requirements.txt
```
3. Create a `.env` file and put the following:
```txt
OPEN_AI_KEY=
INSTAGRAM_USERNAME=
INSTAGRAM_PASSWORD=
```
4. Run `main.py` to test
5. A server should be running on http://localhost:5022/
